Pre-Requisites¶

  • 1 running instance of NetApp Service Level Manager(NSLM).
  • Kubernetes 1.9+ installation

Note

  • All validation in this solution has been done with RHEL 7.3, the source files can run on any flavour of Linux.
  • Any number of nodes can be added in the kubernetes cluster, for demo purposes a 3 node cluster is considered in this validation.
